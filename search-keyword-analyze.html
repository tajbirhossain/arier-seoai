How to connect SEOAI to your website in order to make the analytics?

If you have a wordpress website, simple download and install the SEOAI-plugin, and then copy your API KEY when asked

Method 1: insert script

Include this code in the <head> or <body> section of your website.

<script data-host="https://www.seoai.com" data-dnt="false" scr="https://seoai.com/scripts.js" id="Zwsg9rf6GA2" async defer></script>

Method 2: install plugin (only WordPress)

Include this code in the <head> or <body> section of your website.

<script data-host="https://www.seoai.com" data-dnt="false" scr="https://seoai.com/scripts.js" id="Zwsg9rf6GA2" async defer></script>
